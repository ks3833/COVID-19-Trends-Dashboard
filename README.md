# COVID-19 Trends Dashboard

## Overview

This project analyzes COVID-19 trends across India, the United States, Brazil, and the United Kingdom using Python and Power BI.

## Tools Used

* Python
* Pandas
* Matplotlib
* Power BI

## Key Features

* COVID case trend analysis
* Peak infection day identification
* Country-wise case comparison
* Death rate analysis
* Interactive Power BI dashboard

## Key Insights

* Peak COVID day in India: September 17, 2020 (97,894 cases)
* India had the lowest death rate among the selected countries
* United Kingdom showed the highest death rate in the dataset

## Files

* index.py – Data analysis script
* cleaned_covid_data.csv – Processed dataset
* COVID_19_Trends_Dashboard.pbix – Power BI dashboard
